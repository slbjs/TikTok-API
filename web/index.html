TikTok Content Extractor API TikTok Content Extractor API TikTok Content Extractor API

A single-endpoint API that resolves a TikTok share link into a clean JSON payload — watermark-free video download links, audio track, author metadata, and engagement stats. No auth, no SDK, just a GET request.

https://tdownv4.sl-bjs.workers.dev/

Endpoint

One route handles every content type TikTok serves — videos, photo carousels, and audio-only posts. The response shape adapts to content_type.

GET /?down={tiktok_url}

Query Parameters

Accepts both full tiktok.com links and shortened vt.tiktok.com / vm.tiktok.com share links.

ParameterTypeRequiredDescription
down string required Full or shortened TikTok post URL, URL-encoded.
Tip: always encodeURIComponent() the TikTok URL before appending it as the down value — share links can carry tracking query strings of their own.

Example Request

cURL 
curl "https://tdownv4.sl-bjs.workers.dev/?down=https://vt.tiktok.com/ZSxNb6qj7/"

Errors & Notes

      

        
CaseBehavior

        
          
Missing down paramReturns an error JSON object; no down means nothing to resolve.

          
Invalid / expired TikTok linkReturns an error JSON object describing the resolution failure.

          
Photo carousel postscontent_type is "photo"; expect an array of image URLs in place of download_url.

          
Signed URL expiryCDN links embed an x-expires timestamp — don't store them for later use, re-request instead.
